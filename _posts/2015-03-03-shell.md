### 6. 远程登陆简化脚本
```sh
#!/usr/bin/expect
set timeout 30
spawn ssh zdd@github.com
expect "*assword:"
send "password\r"
interact 
```

### 4. Shell 读两个文件
```sh
#!/bin/sh 

exec 3<$1 
exec 4<$2 
while read line1 <&3 && read line2 <&4 
do 
    echo "$line1" 
    echo "$line2" 
done 
```
